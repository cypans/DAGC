# DAGC--Decomposed Adaptive-hop Graph Convolution for Node Classification 

## Python environment setup with Conda

 
Tested with Python 3.7, PyTorch 2.7, and PyTorch Geometric 2.3.1.


```bash

pip install pandas

pip install scikit_learn

pip install numpy

pip install scipy

pip install einops

pip install ogb

pip install pyyaml

pip install googledrivedownloader

pip install networkx

pip install gdown

pip install matplotlib

```
